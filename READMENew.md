# E-Commerce Sales Analysis

## Project Overview
Comprehensive analysis of e-commerce sales data to uncover insights about revenue trends, customer behavior, and product performance.

## Technologies Used
- Python 3.9+
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Installation
bash
pip install -r requirements.txt


## Usage
bash
python generate_sample_data.py
python main_analysis.py
